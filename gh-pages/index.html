NIfTI Phantoms

A universal, implementation-agnostic format for storing MRI simulation phantoms.

One JSON file defines tissues and their MR properties (T1, T2, …) and references NIfTI files for per-voxel data — both widely supported and easy to view, edit and version. The goal: vary experiments by editing JSON, not code, so phantom data is exchangeable and reproducible.
